Check signature functionality

Mocks the check of a signature using public key and the message Normally in LIGO the message is packed using Bytes.pack (See https://ligolang.org/docs/reference/bytes-reference/) Which converts it into a binary format used for serilization. Since this methods mocks the check of a signature we simply pass the message as the type it is.
